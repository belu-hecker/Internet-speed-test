                                                                  INTERNET SPEED TEST
                                                            --------------------------------

This is a simple GUI application for checking the internet speed! 

To run this program, you need to install tkinter and speedtest module ....

``pip install tkinter``

``pip install speedtest-cli``

To run the application, write this command on terminal:

``python3 Internet.py``

It will open a GUI dialog box. Click on ``Check Speed`` to check the speed of the internet.
